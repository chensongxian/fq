# 科学上网指南
## ss/ssr
> ss/ssr是一群大佬弄的科学上网方式，是目前最稳定最好用的科学上网工具之一

> ss/ssr各个客户端支持都比较好，windows、macOs、ios、安卓均有支持

> 废话不多说，那下面就来介绍各个客户端ss/ssr的使用

## windows
